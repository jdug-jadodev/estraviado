# RutaCo - Guia de Desarrollo y Aprendizaje
## Indice General

Este conjunto de documentos es la guia completa de desarrollo de RutaCo.
Cada archivo cubre una sub-fase especifica con explicaciones tecnicas detalladas,
proposito de cada decision, y contexto de aprendizaje para cualquier desarrollador
que se una al proyecto.

---

## Como leer esta guia

Cada documento sigue esta estructura:
- Que vamos a hacer en esta sub-fase
- Por que lo hacemos asi (decision tecnica)
- Explicacion de cada tecnologia y concepto nuevo
- Codigo paso a paso con comentarios detallados
- Checklist de verificacion al terminar
